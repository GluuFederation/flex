Sample client for Casa credentials enrollment

Note: Check the README before any usage!.

The following lists the supported authentication methods in the API, ensure they are already enabled in the underlying Gluu installation:
  • Verified mobile phones (acr = twilio_sms)
  • OTP apps (acr = otp)
  • Super Gluu (acr = super_gluu)
  • Security keys (acr = fido2)
The following can be used to get/set user 2FA info:
  • User's 2FA info
